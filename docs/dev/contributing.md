# Contributing to PyDigi

Contributions are welcome! This page will be expanded with contribution guidelines.

## Development Setup

```bash
git clone https://github.com/yourusername/pydigi.git
cd pydigi
pip install -e ".[dev]"
```

## Running Tests

```bash
pytest
```

## Building Documentation

```bash
mkdocs build
mkdocs serve  # Preview at http://127.0.0.1:8000
```
